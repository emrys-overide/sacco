# Secretary-confirmed Chairman recovery

This is an exceptional account-recovery process for a Chairman who cannot sign
in. It is not a general Secretary password-reset permission.

1. The Chairman selects **Chairman recovery** on the sign-in page and enters
   their registered phone number or email.
2. The public response is deliberately generic, so it does not reveal whether
   an account exists. A pending request is created only for an active Chairman
   account.
3. Every active Secretary receives a private in-app notification linking to
   **Chairman Recovery**. No other role receives the request.
4. The Secretary verifies the Chairman through the SACCO's agreed process,
   such as an in-person ID check and a trusted registered contact.
5. The Secretary issues an eight-character-or-longer temporary recovery
   password. It expires in 24 hours and the Chairman must replace it before
   using the application.
6. The approval, reviewer, and time are retained in the recovery request and
   audit log.

The Secretary cannot use this authority to reset member, Treasurer,
Accountant, Auditor, or another Secretary account. Ordinary member and officer
password resets remain Chairman-controlled.

If officer MFA is enabled, the recovery password does not bypass the
authenticator-app code. Losing both the password and the authenticator device
requires the SACCO's Technical Department and Board-approved emergency process.
